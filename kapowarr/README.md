## Configuration

This add-on provides the following configuration options:

| Option | Description |
|--------|-------------|
| **Log Level** | Controls the verbosity of log output. Use 'info' for normal operation, 'debug' for troubleshooting, 'warning' or 'error' for minimal output. |
| **Timezone** | The timezone for the application (e.g., 'America/New_York', 'Europe/London'). Defaults to UTC. |
